Process

애자일 2주 스프린트. 매 스프린트마다 TestFlight 빌드를 공유합니다.

유저 리서치, 와이어프레임, 기술 스택 선정. 1주 내 PRD 완성.

FAQ

자주 묻는 질문

hello@appcraft.io 또는 카카오톡 채널로 문의해 주세요.

MVP·스타트업은 React Native/Flutter로 비용·속도 최적화. 고성능 게임, AR/VR, OS 특화 기능이 필요하면 Swift/Kotlin 네이티브를 추천합니다. 상담 시 프로젝트에 맞는 스택을 제안해 드립니다.
MVP 패키지는 평균 6~8주입니다. 기능 범위와 플랫폼 수에 따라 달라지며, Discovery 단계에서 상세 일정을 확정합니다.
프로젝트 완료 시 소스코드, 디자인 에셋, IP 모두 클라이언트에게 이전됩니다. GitHub repo 접근권한도 함께 제공합니다.
iOS App Store, Google Play 심사 경험이 풍부합니다. 리젝 대응, 메타데이터 최적화, ASO까지 Full App 패키지에 포함됩니다.
Full App 패키지에 30일 버그픽스가 포함됩니다. 이후 Retainer(월 ₩5M~) 또는 건별 유지보수 계약이 가능합니다.
네, mid-development 합류, 코드 리뷰, 특정 기능만 개발하는 partial engagement도 가능합니다.
